# 하이웍스 첨부 일괄 다운로드 (v0.1)

## 설치
1. 폴더 저장 → `chrome://extensions` → 개발자 모드 ON → 압축해제된 확장 로드

## 사용
1. 하이웍스 메일 목록을 **한 번 새로고침하거나 스크롤**(필수) → 확장이 목록 응답을 수집
2. 확장 아이콘 클릭 → 팝업 **1) 대상 메일** 목록에 제목들이 뜸 
3. **2) 선택한 메일의 첨부 불러오기**
4. **3) 첨부 선택 후 다운로드**

## 진단창
- `API: 미탐지` → 메일을 한 번 열었다 나오거나 목록 새로고침
- `수집된 메일: 0개` → 하이웍스 목록을 새로고침/스크롤해 목록 API를 태우세요
- 페이지 이동해도 수집된 메일은 팝업에 누적됩니다

## 안 될 때
- **메일이 목록에 안 뜸**: 목록 API 응답의 필드명(제목/ID 키)이 예상과 다를 수 있음.
  하이웍스 목록에서 F12 → Network → 목록 불러올 때 오는 응답(JSON) 하나를
  개발자에게 전달하면 규칙을 맞춰드립니다.
- **첨부를 못 찾음**: F12 콘솔에 `[하이웍스DL] 첨부 못찾음, 응답 원문:` 이 찍힘.
  그 JSON을 전달하면 첨부 추출 규칙을 맞춰드립니다.

## 주의
- 사내 보안/DRM 정책 확인 권장
- 확장 로드/수정 후에는 하이웍스 탭 새로고침 필수
